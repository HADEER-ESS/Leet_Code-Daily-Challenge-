- Difficulty : Easy
- Topic : Linked List, Reverse, Array

- Intuition : 
    - Reverse NODE in the given List

- Approach : 
    - create a pointer list-node to traverse through given array `list` `list= [2 -> 3 -> 4 -> 5 -> 6]`
    - create a list-node to store the reverse list  `rev` init with 0 `rev = [0 ->]`
    - create a pointer list-node to traverse through REV `ptr`
    - create an array to get the value for each node `arr`

    - traverse through `list`, and store the value of each node in `arr` `arr = [2,3,4,5,6]`
    - loop through the `array`, and pop VALUES from it and create a new node linked to `ptr` list-node `rev= [0 -> 6 -> 5 -> 4 -> 3 -> 2]`
    - return `rev.next` `[6 -> 5 -> 4 -> 3 -> 2]`

- Complexity :
    - Time complexity  : O(n)
    - Space complexity : O(n)
