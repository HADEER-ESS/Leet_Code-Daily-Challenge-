- Difficulty : Medium
- Topic : Array, Dynamic Programming

- Intuition : 
    - square may have 1*1 , or 2*2 or n*n ones
- Approach :
    - to check if it is a perfict square when check all its neighbours
        - [r - 1][ c ]
        - [ r ][c - 1]
        - [r - 1][c - 1]

- Complexity :
    - Time complexity : O(n)
    - Space complexity : O(1)
