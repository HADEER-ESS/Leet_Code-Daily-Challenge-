- Difficulty : Medium
- Topic : Hash Table, String, Sorting, Heap (priority Queue), Countring

- Intuition : 
    - resort the string based on every character frequency.

- Approach : 
    - get each character frequency
    - sort the characters based on its frequency
    - try to add every character in a new string
    - return that string

- Complexity :
    -Time complexity : O(n log n)
    - Space complexity : O(n)
