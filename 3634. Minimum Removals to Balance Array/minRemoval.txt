- Difficulty : Medium
- Topic : Array, Sorting, Sliding Window

- Intuition : 
    - make the differance between MAX and MIN value in array apply this equation`max <= min * k`

- Approach : 
    - sort the array
    - loop through the array using two Pointer END always increment and START increment only when
        `nums[end] > nums[start]*k`
    - return START value

    - [1,2,6,9] k=3
    - start = 0
    - end = 0
    - 1 > 1*3 => FALSE => end++;
    - 2 > 1*3 => FALSE => end++;
    - 6 > 1*3 => TRUE  => start++; end++
    - 9 > 2*3 => TRUE  => start++; end++

    - return start = 2


- Complexity :
    - Time complexity  : O(n log n)
    - Space complexity : O(1)
